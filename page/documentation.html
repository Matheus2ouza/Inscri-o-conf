GUIA DO SISTEMA

Objetivo da Documentação

Esta documentação foi criada para ajudar os usuários a realizar suas inscrições de forma correta e eficiente, além de esclarecer dúvidas que possam surgir durante o uso do sistema de Inscrição. O guia também visa fornecer informações práticas sobre o processo de inscrição e pagamento, garantindo que todos os aspectos da inscrição sejam compreendidos. Nosso objetivo é facilitar a experiência do usuário e promover um entendimento claro do sistema.

Caso tenha outras dúvidas, verifique também a Carta Convite.

LOGIN

O nosso novo modelo de sistema de inscrições agora vem com contas separadas para cada localidade, com fim de dar mais segurança e privacidade.

Como fazer login?

O acesso ao sistema é feito por meio de um único usuário por localidade, responsável por inscrever todos os irmãos da sua cidade ou região.

Se você não é o responsável pela inscrição da sua localidade, entre em contato com ele para que realize o processo por você.

Se você é o responsável e ainda não possui login e senha, entre em contato com o suporte. Nossa equipe fornecerá seu acesso ao sistema.

Cada localidade possui apenas um login autorizado para uso.

1. INSCRIÇÃO

Instruções detalhadas sobre como se inscrever corretamente.

1.1 Tipos de Inscrição

O sistema de inscrição oferece dois modos: Inscrição Individual e Inscrição em Grupo. Cada modalidade possui um processo específico. Abaixo, você encontra o passo a passo detalhado para cada tipo de inscrição.

1.2 Inscrição em Grupo

A Inscrição em Grupo permite cadastrar vários participantes de uma só vez. Para que o sistema processe as informações corretamente, os dados devem seguir regras específicas. Veja abaixo o passo a passo para realizar esse tipo de inscrição.

1.2b Dados Iniciais

Primeiramente, selecione o evento no qual deseja realizar a inscrição. Abaixo da data do evento, você poderá verificar os tipos de inscrição disponíveis e seus respectivos valores.

Em seguida, preencha o campo do responsável. Essa pessoa será encarregada pelas inscrições e receberá os identificadores no momento da conferência. Pedimos para que tenha somente uma pessoa responsável por todas as inscrições independente de ser em grupo ou individual

1.2c Dados Requeridos

Para facilitar a inscrição de vários participantes de uma só vez, nenhum dado é preenchido diretamente no sistema. Em vez disso, disponibilizamos uma planilha pré-formatada para que você insira as informações dos participantes. Essa planilha pode ser baixada no momento da inscrição.

Ou se preferir pode baixar clicando abaixo:

1.2d Planilha

A planilha foi previamente estruturada para facilitar o preenchimento e a leitura dos dados durante o processo de inscrição. Cada campo exige informações em um formato específico. Veja abaixo as instruções para o preenchimento:

1. Nome: Nome completo, sem números ou caracteres especiais.

2. Data de nascimento: No formato DD/MM/AAAA.

3. Sexo: Apenas Masculino ou Feminino.

4. Tipo de inscrição: Deve corresponder a uma das opções disponíveis no card do evento selecionado no passo anterior.

Qualquer dos campos que forem preenchidos de forma errada resultarar em um erro no sistema e não será feita a inscrição

1.2f Lista e Confirmação

Caso não sejam encontrados erros na planilha, será carregada uma lista com os dados de cada participante. Verifique com atenção se todas as informações estão corretas antes de prosseguir.

Após confirmar os dados, serão exibidas as informações gerais da inscrição que está sendo realizada. Revise com atenção todos os detalhes e, se estiverem corretos, confirme a inscrição. A inscrição será registrada no sistema e gerará um débito para a localidade, que deverá ser pago até a data limite.

1.3 Inscrição Individual

A inscrição individual segue o mesmo princípio em relação ao preenchimento dos dados do responsável e à seleção do evento. Caso tenha dúvidas sobre essas etapas, consulte o tópico 1.2b.

1.3b Dados do participante

Ao contrário da inscrição em grupo, na inscrição individual os dados do participante são preenchidos diretamente no site, seguindo as mesmas regras definidas na planilha.

1. Nome: Nome completo, sem números ou caracteres especiais.

2. Data de nascimento: No formato DD/MM/AAAA.

3. Sexo: Apenas Masculino ou Feminino.

4. Tipo de inscrição: Deve corresponder a uma das opções disponíveis no card do evento selecionado no passo anterior.

1.3b Confirmação de Inscrição

Após confirmar os dados, serão exibidas as informações gerais da inscrição que está sendo realizada. Revise com atenção todos os detalhes e, se estiverem corretos, confirme a inscrição. A inscrição será registrada no sistema e gerará um débito para a localidade, que deverá ser pago até a data limite.

2. PAGAMENTO

Cada inscrição Os valores referentes a inscrição individual ou em grupo, seram todos ligados a localidade e teram que ser quitados até a data limite

2.1 Valores

Cada evento possui seus próprios tipos de inscrição e respectivos valores. Por isso, ao realizar uma inscrição, verifique qual tipo é válido para o evento selecionado. Assim que uma inscrição é concluída, o valor correspondente é adicionado ao saldo da localidade. Por exemplo, se uma localidade possui um saldo devedor de R$ 200,00 e realiza uma nova inscrição no valor de R$ 300,00, o saldo devedor será atualizado para R$ 500,00.

2.2 Valores por Inscrição

Cada inscrição tambem tem seu saldo devedor:

Uma inscrição só será validada ao final do prazo se o saldo devedor estiver totalmente quitado. No caso de inscrições em grupo, onde há múltiplos participantes, é possível realizar pagamentos parciais. No entanto, o valor total da inscrição deve ser pago até a data limite para que todos os inscritos sejam considerados oficialmente registrados como pagos.

2.3 Pagamentos

Todas as inscrições devem estar com os pagamentos quitados até a data limite. No nosso sistema, o pagamento não é realizado diretamente — apenas o comprovante de pagamento deve ser anexado.

São aceitos comprovantes nos formatos PDF, PNG ou JPG. Comprovantes fora desses formatos não serão armazenados pelo sistema e resultarão em erro.

Os valores gerados por cada inscrição não precisam ser pagos em uma única transação. Por exemplo, uma inscrição em grupo que gerou um saldo devedor de R$ 600,00 pode ter dois comprovantes de R$ 300,00 cada. O que é exigido é que o valor total do saldo devedor seja quitado até a data limite.
